# dhivatar
PHP port of phoenixatom Dhivatar package


## Installation

```
composer require jinas/dhivatar
```

## Usage

```php
(new Dhivatar)->create()
            ->setText("އަހުމަދު")
            ->output();
```

You can override any defaults configs by passing params.

I will add more configuration options later when I get time.
